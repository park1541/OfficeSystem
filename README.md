# OfficeSystem (사내 업무 관리 시스템)

ASP.NET Core 기반 사내 업무 관리 시스템입니다.
사용자 관리·권한·전자결재를 중심으로, 실제 사내에서 쓰는 업무 시스템을 목표로 개발 중입니다.

## 개발 동기
C#/.NET 기반 백엔드 개발 역량을 키우기 위해 시작한 개인 프로젝트입니다.
기존에 Java/Spring Boot로 다뤘던 개념(인증, 권한, ORM)을 .NET 환경으로
옮기며 양쪽을 비교 학습하는 것을 목표로 합니다.

## 기술 스택
- **Backend**: C# / ASP.NET Core (Web API)
- **ORM**: EF Core
- **Database**: MSSQL
- **Auth**: ASP.NET Core Identity
- **형상관리**: Git / GitHub

## 주요 기능
- [ ] 로그인 / 인증
- [ ] 사용자(사원) 관리
- [ ] 권한 / 역할 관리
- [ ] 전자결재 (상신 → 승인 / 반려)
- [ ] 대시보드 *(예정)*
